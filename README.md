## Procedure to Run the Spatial Filtering Project

### **For Python Version (spatial_filtering.py)**

**Prerequisites:**
1. Python installed (Python 3.x recommended)
2. Required libraries:
   ```
   pip install opencv-python numpy matplotlib
   ```

**Steps to Run:**
1. Navigate to the project directory:
   ```
   cd "d:\Nishanth CV project"
   ```

2. Ensure sample.jpg exists in the project directory (it should be present)

3. Run the script:
   ```
   python spatial_filtering.py
   ```

4. A matplotlib window will appear showing:
   - **Original Image** (left)
   - **Mean Filter Result** (middle) - 5×5 averaging kernel
   - **Gaussian Filter Result** (right) - 5×5 Gaussian kernel

---

### **For Java Version (SpatialFiltering.java)**

1. Compile:
   ```
   javac SpatialFiltering.java
   ```

2. Run:
   ```
   java SpatialFiltering
   ```

---

### **For MATLAB Version (spatial_filtering.m)**

1. Open MATLAB
2. Navigate to the project directory
3. In MATLAB command window, type:
   ```
   spatial_filtering
   ```

---

**Note:** The Python version is the easiest to run and will display the filtered images in an interactive window.
